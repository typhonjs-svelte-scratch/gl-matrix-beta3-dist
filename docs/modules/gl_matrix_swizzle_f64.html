gl-matrix (Node / NPM Distribution)

    Package gl-matrix/swizzle/f64

    To enable additional swizzle accessors for vector classes (64-bit) invoke the EnableSwizzlesF64 function from the gl-matrix/swizzle/f64 sub-path export. To enable ambient module declarations for IDE / Typescript support please see gl-matrix/types/swizzle/f64.

    To enable swizzling for the 32-bit variation of gl-matrix please see gl-matrix/swizzle.

    Example

    import { Vec3 } from 'gl-matrix/f64';
    import { EnableSwizzlesF64 } from 'gl-matrix/swizzle/f64';

    EnableSwizzlesF64();

    const vec = new Vec3(0, 1, 2);
    const vecSwizzled = vec.zyx; // Returns a new Vec3(2, 1, 0).
